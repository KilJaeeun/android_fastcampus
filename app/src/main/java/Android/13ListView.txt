

1. 리스트로 만들고 싶은 아이템의 리스트를 준비한다.
2. 인플레이터를 준비한다.
3. 인플레이터로 아이템 하나에 해당하는 뷰를 만들어 준다.
4. 위에서 만든 뷰를 컨테이너 뷰에 붙여준다.


ListView
1. 리스트로 만들고 싶은 아이템의 리스트를 준비한다.
2. Adapter 을 이용한다. (위의 234단계를 포함하는 행위)


AddView 와 ListView 의 차이점

1. 만드는 방식이 다르다.
2. 그리는 방식이 다르다.

    - AddView -> 리스트의 개수와 상관없이 한 번에 다 그린다.
    - ListView ->  보여지는 부분+ 알파만 한번에 그리고 필요한 경우에 더 그린다.
